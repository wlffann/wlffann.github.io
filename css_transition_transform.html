CSS Transition & Transform

Or, How To Make Any Webpage More Awesome

If you are a new web developer like me, you're always looking for ways to make your page more interesting without having to wrestle with CSS for hours. If your project has an extensive back-end, it can be so frustrating to waste time pushing div elements around the browser just to make your page readable.


Have no fear! Even you can make a boring Bootstrap site exciting by adding two simple properties: transition and transform. These two elements will give you control to move or change any selector or pseudo-selector on your page. Letâ€™s take a closer look at each.


transition

Transition is a great place to start. The way that we define our transitions will have the greatest effect on the look and feel of our interactions.


transition is actually a shorthand property for these four other properties:

  • transition-delay: controls any pause time before the transition begins. Default value of 0s.
  • transition-duration: sets the time that the transition takes to execute. This duration begins after the delay time passes. Default value of 0s.
  • transition-property: indicates the properties in the pseudo-selector to apply the transition to. Default value of all.
  • transition-timing-function: provides a mathematical function to change the speed of the transition at different points int the duration. Default value of ease.

All of these properties can be included in the shorthand transition property in the format below:

transtition: property duration timing-function delay;


transform

Transformations let you define the actual change you want to see in the element. This means you can make a simple div grow, shrink, move left, or spin 360deg depending on the interaction that you want to have with your user.


Some fun options include:

  • scale() changes the dimensions of the element by multiplying the size by the given value. Takes an integer value.
  • skewY() stretches the outer edges of the element vertically or horizontally to create a parallelogram. Takes a degree value.
  • translate() moves the element vertically or horizontally
  • rotate() spins the element clockwise. Takes a degree value.


These functions are nested in the transform property inside of the element's pseudo-selector:

transform: scale(1.5);

The transform-origin property gives you control over the point at which the transformation centers around. It defaults to the center of the element.
